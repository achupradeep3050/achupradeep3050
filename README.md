## 📈 Smart Money Concepts — My Trading Philosophy

<div align="center">

```
FAIR VALUE GAP (FVG) STRATEGY - XAUUSD

STEP 1 — TREND BIAS (EMA 20 / EMA 50)
  EMA20 > EMA50 rising   =>  BULLISH - Look for Bullish FVGs
  EMA20 < EMA50 falling  =>  BEARISH - Look for Bearish FVGs
  Transitioning          =>  NEUTRAL - No trades placed

STEP 2 — FVG DETECTION (3-candle imbalance)
  C1 -> C2 (impulse) -> C3
  Gap = space between C1.high and C3.low (bullish FVG)
  Entry at FVG midpoint  |  SL at swing  |  TP at 2R (1:2 R:R)

STEP 3 — EXECUTION (every 15 seconds)
  Multi-broker: OANDA (Forex) + TOOBIT (Futures)
  Auto lot-sizing with risk-adjusted position size
```
